# VBA_Wind_Turbine_Reporting_Automation

## Project Overview

This project is an Excel VBA application designed to automate operational reporting processes.

The system enables users to generate multiple reports from predefined templates, import information from Word documents, manage reporting data, and create summary reports for analysis.

## Tools

* Microsoft Excel
* VBA (Visual Basic for Applications)
* Microsoft Word Automation

## Key Features

### Report Generation

* Selection of report templates
* Batch creation of multiple reports
* Automatic population of templates with source data
* Automated file naming and saving

### Data Import

* Import of information from Word documents
* Extraction and processing of operational data
* Consolidation of imported information into Excel tables

### Data Management

* User-friendly VBA forms
* Input validation
* Automated formatting
* Structured data storage

### Reporting & Analysis

* Automatic calculation of work durations
* Pivot table generation
* Summary reports for operational analysis
* Time tracking by turbine and activity

## Business Impact

* Reduced manual reporting effort
* Improved reporting consistency
* Minimized human errors
* Accelerated report preparation
* Enhanced operational data analysis

## Technical Notes

This project is based on a real-world reporting solution.

All sample data, company names, turbine identifiers, and operational records have been anonymized and replaced with fictional values to protect confidential business information.

The solution was developed and tested using the Ukrainian version of Microsoft Excel. Minor localization adjustments may be required for other language versions.
